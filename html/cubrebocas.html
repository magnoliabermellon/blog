¿Cómo hacer un cubrebocas casero?

Materiales:

1. Molde de cubrebocas (Descargalo aquí)

2. Hilo para coser

3. Una tela para filtro.

4. Una tela para la textura del cubrebocas.

5. Una tela para repeler el agua.

Instrucciones:

1. Dibuja o imprime el molde.

2. Cose las telas para que sean una sola.

3. Ajusta el molde (deja 1cm para la costura).

4. Corta la forma del cubrebocas (vas a tener 2 piezas).

5. Cose.

6. Ya te lo puedes poner.
